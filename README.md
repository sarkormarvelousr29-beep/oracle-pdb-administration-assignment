Oracle Pluggable Database (PDB) Administration Assignment

Overview

This repository contains my Oracle Pluggable Database (PDB) Administration Assignment completed as part of my Database Administration coursework. The assignment demonstrates practical skills in creating, configuring, managing, and administering Oracle Pluggable Databases (PDBs) using Oracle Database 21c and Oracle SQL Developer.

Objectives

The objectives of this assignment are to:

- Understand the Oracle Multitenant Architecture.
- Create and manage Pluggable Databases (PDBs).
- Perform PDB administration tasks.
- Configure users and privileges.
- Execute SQL commands to verify database operations.
- Gain hands-on experience with Oracle Database 21c.

Tools Used

- Oracle Database 21c
- Oracle SQL Developer
- SQL
- Git & GitHub

Assignment Contents

- Oracle Database connection setup
- Creation of a Pluggable Database (PDB)
- Opening and closing the PDB
- Saving the PDB state
- Creating administrative users
- Granting privileges
- Verifying database configurations
- SQL scripts and execution results
- Screenshots demonstrating successful completion

Learning Outcomes

By completing this assignment, I learned how to:

- Work with Oracle Multitenant Architecture.
- Create and manage Pluggable Databases.
- Administer Oracle database users and privileges.
- Execute Oracle SQL commands using SQL Developer.
- Document and present database administration tasks professionally.
